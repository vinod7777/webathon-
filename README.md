# StockFlow - Inventory Management System

A comprehensive inventory management application for tracking products, sales, and stock levels.

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Development Setup

Clone the repository and install dependencies:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

### Edit via IDE

You can also edit files directly in your preferred code editor or GitHub:
- Clone the repository, make changes locally, and push to update the codebase
- Edit files directly in GitHub by clicking the "Edit" button (pencil icon)
- Use GitHub Codespaces for cloud-based development

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build and deploy the application:

```sh
npm run build
```

Host on your preferred platform (Vercel, Netlify, AWS, etc.) or your own server.

## Custom Domain

To connect a custom domain to your application, configure DNS settings on your hosting provider to point to your deployed instance.
